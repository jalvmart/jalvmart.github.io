HTLM y CSS Básico

Conceptos básicos de HTML: HTML ( Hyper T ext M arkup L anguage ) es el código que se utiliza para estructurar una página web y su contenido. Por ejemplo, el contenido podría estructurarse dentro de un conjunto de párrafos, una lista de puntos con viñetas o utilizando imágenes y tablas de datos. Como sugiere el título, este artículo le dará una comprensión básica de HTML y sus funciones.

Estructura del documento y del sitio web: Además de definir partes individuales de su página (como "un párrafo" o "una imagen"), HTML también cuenta con una serie de elementos a nivel de bloque que se usan para definir áreas de su sitio web (como "el encabezado", "la barra de navegación"). menú", "la columna de contenido principal"). Este artículo analiza cómo planificar la estructura básica de un sitio web y escribir el HTML para representar esta estructura.

Modelo de Caja

Introducción al modelo de caja básica CSS: Al diseñar un documento, el motor de representación del navegador representa cada elemento como un cuadro rectangular de acuerdo con el modelo de cuadro básico CSS estándar . CSS determina el tamaño, la posición y las propiedades (color, fondo, tamaño del borde, etc.) de estos cuadros.

Diseño y el bloque contenedor: El tamaño y la posición de un elemento a menudo se ven afectados por el bloque que lo contiene . La mayoría de las veces, el bloque contenedor es el área de contenido del ancestro más cercano a nivel de bloque de un elemento, pero no siempre es así. En este artículo, examinamos los factores que determinan el bloque contenedor de un elemento.

Introducción a la cascada de CSS: La cascada se encuentra en el núcleo de CSS, como lo enfatiza el nombre: hojas de estilo en cascada. Cuando un selector coincide con un elemento, se aplica el valor de propiedad del origen con la prioridad más alta, incluso si el selector de un origen o capa de menor precedencia tiene mayor especificidad.

Cascada y Herencia

Cascada y herencia: El objetivo de esta lección es desarrollar su comprensión de algunos de los conceptos más fundamentales de CSS: la cascada, la especificidad y la herencia, que controlan cómo se aplica CSS a HTML y cómo se resuelven los conflictos entre las declaraciones de estilo.

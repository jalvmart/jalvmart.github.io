Reseña del programa

¡La Web está en todas partes! El desarrollo web es un trabajo intrincado, creativo e interesante que puede conducir a una carrera emocionante en un campo de rápido crecimiento y demanda.

En este programa de Certificado Profesional, diseñado por el W3C (World Wide Web Consortium), creador de los estándares web, aprenderá todas las habilidades necesarias para crear experiencias de usuario interactivas y receptivas en la web. Este programa profundizará su conocimiento de los 3 lenguajes fundamentales que impulsan la Web: HTML5, CSS y JavaScript. ¡Se le guiará, paso a paso, sobre cómo utilizar todas las técnicas de desarrollo web del lado del cliente para crear e innovar en la web!

Cada curso también incluirá ejemplos interactivos para ayudarlo a practicar y codificar de la manera correcta.
